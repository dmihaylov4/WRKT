## üí° Advanced Tips

### **1. Plurals Don't Matter**
Apple automatically handles singular/plural:
- `workout` = also matches "workouts"
- `challenge` = also matches "challenges"
- Save characters by using singular

### **2. Word Order Doesn't Matter**
Apple creates all combinations:
- `fitness,workout,tracker` = matches "fitness tracker", "workout tracker", "fitness workout tracker"

### **3. Localization Is Free Visibility**
Translating keywords gets you:
- Separate keyword sets per country (100 chars each!)
- Access to non-English markets
- Lower competition in some languages

### **4. Update Keywords Often**
- You can change keywords with each app update
- Test seasonal keywords ("summer", "beach", "shred") in June-July
- Test New Year keywords ("resolution", "goals") in December-January

### **5. Use Search Ads Data**
- Run small Search Ads campaigns ($50-100)
- See which keywords convert best
- Move winning keywords to organic keyword set

## üìû Questions?

**"Should I include 'free'?"**
No. Apple doesn't rank apps based on price, and it wastes characters.

**"Should I include competitor names?"**
No. It's against Apple's guidelines and won't help you rank.

**"How often should I update keywords?"**
Every 2-3 months, or when you add major features.

**"Can I see which keywords are working?"**
Yes, in App Store Connect ‚Üí Analytics ‚Üí Sources ‚Üí App Store Search

**"Should I use the same keywords in my app description?"**
No. Keywords are separate. Use natural language in your description.
