## üöÄ Quick Start - How to Use These Files

### **Step 1: Open App Store Connect**
1. Go to https://appstoreconnect.apple.com
2. Sign in with your Apple ID
3. Click "My Apps"
4. Select WRKT (or create a new app if first submission)

### **Step 2: Upload Screenshots**
1. Go to your app ‚Üí Version ‚Üí Screenshots
2. Select "6.7" Display" (iPhone 14 Pro Max)
3. Drag and drop all 10 screenshots **in numerical order**
4. Verify they appear in the correct sequence

### **Step 3: Add App Description**
1. Go to App Information ‚Üí Description
2. Copy all text from `app_store_description.txt`
